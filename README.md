# Decentralized Donation System

A blockchain-based donation platform built using Solidity, Ethereum, Truffle, and Ganache.

## Project Idea
The system allows users to send donations through a smart contract while maintaining transparency, security, and immutable transaction records.

## Features
- Secure Donation Processing
- Smart Contract Deployment
- Donation Event Tracking
- Total Donation Monitoring
- Blockchain Transaction History
- Owner Withdrawal Function

## Technologies Used
- Solidity
- Ethereum
- Truffle Framework
- Ganache
- JavaScript
- Blockchain Technology

## Workflow
Donation → Smart Contract → Blockchain Recording → Event Emission → Transaction Tracking

## Applications
- Charity Platforms
- Crowdfunding Systems
- Non-Profit Organizations
- Transparent Donation Systems
